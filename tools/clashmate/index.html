手动添加节点

可手动添加节点（支持 dialer-proxy 配置代理链）。添加后会出现在节点列表中。

2

自定义代理组

默认已创建 Auto 组（包含所有节点）。可添加自定义分组。

📖 代理组类型说明
url-test
自动测速选择延迟最低的节点。定期向指定 URL 发请求，自动切换到最快的可用节点。
select
手动选择。用户在客户端手动选择使用哪个节点，不会自动切换。
fallback
故障转移。按列表顺序使用第一个可用节点，当前节点不可用时自动切换到下一个。
load-balance
负载均衡。将请求分散到多个节点，可以选择轮询或一致性哈希策略。
